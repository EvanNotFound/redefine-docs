# footer

```yaml
footer:
  since: 2020      # the starting year of your website, Can be null
  icp:             # ICP record number of your website, Can be null
```

该配置项用于设置网站底部的信息展示。

- `since` 设置网站起始年份，可以为空。
- `icp` 设置网站 ICP 备案编号，可以为空。
