# 快速开始

在使用 Redefine 主题之前，请你先阅读 「 Easy Hexo 团队 」撰写的 [Hexo 教程](https://easyhexo.com/)，了解前置知识再来安装


## 安装

- 如果你的 Hexo 版本在 `5.0` 及以上，推荐通过 `npm` 安装

  ```sh
  $ cd your-hexo-site
  $ npm install hexo-theme-redefine
  ```

- 或者使用 git 克隆

  ```sh
  $ cd your-hexo-site
  $ git clone https://github.com/EvanNotFound/hexo-theme-redefine.git themes/redefine
  ```

## 使用

安装完成后，在 Hexo 配置文件中将 `theme` 设置为 `Redefine`。

```yml
theme: redefine
```

## 更新

Theme Redefine 经常发布新版本，你可以通过如下命令更新 Theme Redefine。

- 通过 `npm` 安装最新版本：

  ```sh
  $ cd your-hexo-site
  $ npm update hexo-theme-redefine
  ```

- 通过 `git` 更新到最新的 `main` 分支：

  ```sh
  $ cd themes/redefine
  $ git pull
  ```
